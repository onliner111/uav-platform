# 管理员操作手册

版本：V1.0\
生成时间：2026-02-21 05:20:56

------------------------------------------------------------------------

# 一、管理员职责

管理员负责：

-   用户与角色管理
-   权限分配
-   审批流程管理
-   数据导出与审计查看
-   系统基础配置

------------------------------------------------------------------------

# 二、用户管理

路径：系统管理 → 用户管理

功能包括：

-   新增用户
-   编辑用户信息
-   启用/禁用账号
-   分配角色

------------------------------------------------------------------------

# 三、角色与权限管理

系统支持以下权限类别：

-   inspection:read / inspection:write
-   defect:read / defect:write
-   emergency:manage
-   reporting:export
-   admin:full

管理员可按部门分配角色。

------------------------------------------------------------------------

# 四、审批管理

路径：系统管理 → 审批管理

支持：

-   审批记录查看
-   审批状态修改
-   审计留痕查询

------------------------------------------------------------------------

# 五、数据导出与审计

路径：报表中心 → 审计导出

支持：

-   操作日志导出
-   问题流转记录导出
-   审批记录导出

------------------------------------------------------------------------

# 六、安全管理

-   定期修改管理员密码
-   定期备份数据库
-   控制外网访问权限

------------------------------------------------------------------------

# 七、常见问题

1.  用户无法登录 → 检查是否被禁用
2.  无法查看数据 → 检查租户权限
