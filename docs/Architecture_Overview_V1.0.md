# 城市无人机巡查与应急指挥平台

# 系统架构说明书

版本：V1.0\
生成时间：2026-02-21 05:20:56

------------------------------------------------------------------------

# 一、总体架构

系统采用：

-   单体架构（Monolith）
-   插件式 Adapter 设计
-   Docker 容器化部署

------------------------------------------------------------------------

# 二、技术栈

-   后端：FastAPI
-   数据库：PostgreSQL + PostGIS
-   前端：Jinja2 + 静态JS
-   容器化：Docker Compose

------------------------------------------------------------------------

# 三、核心模块

1.  巡查管理模块
2.  问题闭环模块
3.  应急指挥模块
4.  指挥大屏模块
5.  报表统计模块
6.  审批与审计模块

------------------------------------------------------------------------

# 四、数据模型

核心数据表包括：

-   inspection_tasks
-   inspection_observations
-   defects
-   incidents
-   approval_records

所有数据均包含 tenant_id 实现租户隔离。

------------------------------------------------------------------------

# 五、安全机制

-   权限控制（RBAC）
-   租户隔离
-   审计日志
-   操作留痕

------------------------------------------------------------------------

# 六、扩展能力

-   支持 DJI / MAVLink 对接
-   支持多租户扩展
-   支持云端部署
