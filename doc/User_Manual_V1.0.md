# 1. 系统简介

本系统用于支撑城市管理部门开展：

-   日常巡查执法\
-   问题闭环管理\
-   应急事件快速处置\
-   指挥调度与统计汇报

系统实现巡查任务创建、问题生成、流转闭环、应急响应和数据统计等功能。

适用对象：

-   城管巡查人员\
-   执法中队长\
-   应急值班人员\
-   指挥中心人员

# 2. 系统登录

## 2.1 访问地址

在浏览器中输入系统部署地址，例如：

http://localhost:8000

正式环境以运维部门提供的地址为准。

------------------------------------------------------------------------

## 2.2 登录流程

1.  输入账号\
2.  输入密码\
3.  点击【登录】

登录成功后进入系统首页。

------------------------------------------------------------------------

# 3. 巡查业务操作指南

## 3.1 创建巡查任务

菜单路径：

巡查管理 → 新建巡查任务

操作步骤：

1.  选择巡查模板（违建/占道经营/垃圾堆放等）\
2.  在地图上圈选巡查区域\
3.  填写任务名称\
4.  点击【创建】

任务状态为：SCHEDULED（已排程）。

------------------------------------------------------------------------

## 3.2 执行巡查

无人机执行飞行任务时：

-   系统自动接收巡查成果\
-   自动记录巡查轨迹\
-   自动生成巡查点位

------------------------------------------------------------------------

## 3.3 查看巡查成果

菜单路径：

巡查管理 → 任务详情

可查看：

-   巡查时间\
-   点位坐标\
-   严重等级\
-   现场图片\
-   备注信息

------------------------------------------------------------------------

## 3.4 导出巡查记录

在任务详情页面点击【导出记录】，生成巡查报告文件。

------------------------------------------------------------------------

# 4. 问题闭环管理

## 4.1 生成问题单

在巡查成果页面点击【生成问题单】，填写问题描述、严重等级、指派人员并提交。

------------------------------------------------------------------------

## 4.2 问题流转状态

问题状态流程：

OPEN → ASSIGNED → IN_PROGRESS → FIXED → VERIFIED → CLOSED

------------------------------------------------------------------------

# 5. 应急指挥操作指南

## 5.1 创建应急事件

菜单路径：

应急管理 → 新建事件

填写事件名称、等级及位置后点击【创建】。

------------------------------------------------------------------------

## 5.2 一键创建应急巡查任务

在事件详情页点击【一键创建巡查任务】，系统自动生成高优先级巡查任务。

------------------------------------------------------------------------

# 9. 常见问题处理

## 地图无法加载

-   检查网络\
-   检查浏览器设置

## 无法创建任务

-   检查是否具备权限\
-   联系管理员
