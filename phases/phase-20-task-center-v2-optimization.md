# Phase 20 - 任务中心 v2 优化

## 0. Basis
- Based on: `111.md`
- Depends on: `phases/phase-19-real-device-video-integration.md`

## 1. Objective
在现有任务中心基础上做产品级增强，提升调度效率、可解释性与冲突治理能力。

## 2. Scope
- 航线模板与载荷配置模板增强
- 自动派单评分模型优化与可解释输出
- 资源匹配与冲突检测优化
- 任务资料复用与模板化编排增强

## 3. Out of Scope
- 全自动无人值守调度闭环
- 外部优化引擎重度依赖

## 4. Deliverables
- 任务模板增强模型与 API
- 派单评分策略与解释字段
- 冲突检测规则与提示机制
- 调度效率与质量回归测试

## 5. Acceptance
- 任务模板可复用率明显提升
- 自动派单结果有可解释评分明细
- 冲突检测可提前拦截高风险排程
- 核心任务链路稳定、无跨租户越权

## 6. Exit Criteria
- `ruff`, `mypy`, `pytest`, `e2e` 全绿
- 任务中心 v2 演示脚本可复跑

---

## 7. Priority Tuning (P0/P1/P2)

- P0（先做，阻塞调度升级）：
  - `20-WP1` 模板增强（航线/载荷）最小闭环
  - `20-WP2` 自动派单评分与解释最小链路
- P1（随后完成，形成运营价值）：
  - `20-WP3` 冲突检测与预警优化
  - `20-WP3` 模板复用与批量任务增强
- P2（可延后到本阶段后半）：
  - `20-WP2` 多目标调度优化与策略对比
- 执行顺序：`P0 -> P1 -> P2 -> 20-WP4`

- [ ] 20-WP2 自动派单评分与解释
- [ ] 20-WP3 冲突检测与任务编排增强
