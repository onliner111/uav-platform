# Phase 31 - 可观测与可靠性运营控制台

## 0. Basis
- Based on: `项目最终目标.md`
- Depends on: `phases/phase-30-commercial-platform-ops-ui.md`

## 1. Objective
把可观测与可靠性能力产品化为运营控制台，支撑值守、巡检、演练、复盘的可视化流程。

## 2. Scope
- 可观测工作台：日志/指标/追踪总览与过滤
- SLO 运营：SLO 状态、异常趋势、告警联动
- 备份恢复运营：备份任务、恢复演练、结果记录
- 安全巡检：巡检任务、报告与整改追踪
- 容量与弹性：容量趋势、预测与策略视图

## 3. Out of Scope
- 跨地域多活架构改造
- 基础设施重构

## 4. Deliverables
- 可观测与可靠性运营控制台页面集
- 值守与演练操作流程 UI
- 巡检与容量运营视图
- 运维运营演示脚本

## 5. Acceptance
- 运维人员可在 UI 进行日常值守与演练
- SLO 与告警联动在 UI 清晰可见
- 巡检与恢复记录可查询可审计

## 6. Exit Criteria
- `ruff`, `mypy`, `pytest`, `e2e` 全绿
- 可观测与可靠性演示脚本可复跑
