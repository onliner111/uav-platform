# Phase 30 - 商业化与平台运营 UI

## 0. Basis
- Based on: `项目最终目标.md`
- Depends on: `phases/phase-29-data-ai-governance-ui.md`

## 1. Objective
建立面向租户管理员与平台运营方的商业化运营 UI，包括计费、配额、租户治理与开放平台接入管理。

## 2. Scope
- 计费中心：套餐、订阅、账单、状态生命周期
- 配额中心：用量趋势、配额阈值、超限预警
- 租户运营：租户视图、状态、基础运营指标
- 开放平台：凭证、Webhook、回调日志与安全设置

## 3. Out of Scope
- 可观测与可靠性深度运营台（Phase 31）
- 架构层改造

## 4. Deliverables
- 计费与配额运营页面
- 租户运营与开放平台页面
- 商业化演示脚本与操作手册更新

## 5. Acceptance
- 租户管理员可在 UI 完成订阅与配额管理
- 平台运营可在 UI 查看租户运营状态
- 对外集成配置可在 UI 完成并可审计

## 6. Exit Criteria
- `ruff`, `mypy`, `pytest`, `e2e` 全绿
- 商业化运营演示脚本可复跑
